# fundraising-strategy

VC戦略立案、投資家関係構築、資金調達実行のための戦略スキル。

## Description

Fundraising Lead向けスキル。VC戦略立案フレームワーク、投資家リスト管理、Pitch Deck戦略、Warm intro発見を提供。

## When to Use

- Pre-seed/Seed調達準備時
- VCターゲティング戦略立案
- Pitch Deck戦略設計
- 投資家リレーションシップ構築
- Warm intro経路発見

## Core Frameworks

### 1. VC Targeting Framework

**基準:**
- **Thesis Fit:** プロトコルの投資テーマと合致するか
- **Stage:** Pre-seed/Seed投資実績
- **Geography:** アジア/グローバル
- **Portfolio:** 類似案件への投資履歴
- **Warm Intro:** イントロ経路の有無

**優先順位:**
1. Thesis Fit + Warm Intro
2. Thesis Fit + Portfolio Match
3. Portfolio Match only
4. Cold outreach

### 2. Pitch Strategy Framework

**構成:**
1. **Problem:** DeFi UXの複雑さ（明確な痛み）
2. **Solution:** Axis Pizzaのシンプル化
3. **Traction:** 127 users（初期検証）
4. **Market:** Solana DeFi成長（タイミング）
5. **Team:** Muse + Superteam JP
6. **Ask:** $750K Pre-seed

**ストーリーテリング:**
- VCの投資テーマに合わせてカスタマイズ
- 類似ポートフォリオ企業を引用
- データよりナラティブ（crypto特性）

### 3. Warm Intro Playbook

**経路発見:**
1. LinkedIn 2nd degree connections
2. Superteam JP network
3. Portfolio company founders
4. Accelerator alumni
5. Crypto Twitter mutuals

**イントロ依頼:**
- 依頼相手: 信頼関係が強い人
- タイミング: VCとの会話後（興味確認後）
- フォーマット: "〇〇さん、△△VCの□□さんとお話ししたいです。紹介いただけますか？"

### 4. Investor List Management

**構造:**
```
VC Database:
- Name
- Thesis
- Portfolio (類似案件)
- Contact (Partner名)
- Warm Intro Path
- Status (Not Contacted / Intro Requested / Meeting Set / Passed / Invested)
- Last Update
```

**更新頻度:** 週次

## Workflow Example

```
1. VCリスト作成
   → VC Researcher (Haiku) に50社調査依頼
   
2. Thesis Fit分析
   → ポートフォリオと投資テーマ確認
   
3. Warm Intro経路発見
   → Intro Hunter (Haiku) にLinkedIn調査依頼
   
4. Pitch Deck作成
   → Pitch Deck Builder (Haiku) にスライド作成依頼
   
5. アプローチ開始
   → 優先順位順にWarm intro依頼
```

## Communication Style

**報告例:**
- "Multicoin invested $2M in Marginfi (similar DeFi), thesis fit confirmed"
- "3 warm intros secured via Superteam JP network"
- "15 VCs targeted this week, 5 meetings set"

**原則:**
- Relationship First（関係構築が第一）
- Thesis Alignment（テーマ合致重視）
- Persistent Follow-up（フォローアップが80%）
- Data-driven targeting（データ駆動）

## Success Metrics

- **VC会議数:** 週5-10件
- **Warm intro率:** 70%以上
- **Thesis fit率:** 80%以上
- **Response rate:** 30%以上（Warm intro）
- **Meeting→Term sheet:** 10-20%
