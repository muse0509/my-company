# Tax Department - 税理士室

## 担当領域

1. **税務処理**
   - 日々の経費の税務分類
   - 控除対象項目の整理
   - 税額計算

2. **確定申告準備**
   - 確定申告書類の準備
   - 必要書類のチェックリスト
   - 申告スケジュール管理

3. **節税対策**
   - 経費の最適化
   - 控除の最大化
   - 税制優遇措置の活用

4. **税務レポート**
   - 月次税務サマリー
   - 四半期税務レビュー
   - 年次税務計画

## 日本の税制対応

### 主な税目

1. **所得税**
   - 青色申告 vs 白色申告
   - 所得控除（基礎控除、社会保険料控除等）
   - 税率区分

2. **消費税**
   - インボイス制度対応
   - 課税事業者判定
   - 仕入税額控除

3. **法人税（将来的）**
   - 法人設立時の検討
   - 法人成りのタイミング
   - 税率比較

### 経費の分類基準

| 分類 | 項目例 | 控除可否 |
|------|--------|---------|
| 開発費 | サーバー代、API利用料、ツール代 | ✅ 全額 |
| 通信費 | インターネット、携帯電話 | ✅ 按分（業務割合） |
| 交通費 | 出張、打ち合わせ | ✅ 実費 |
| 交際費 | VCとの食事、イベント | ✅ 業務関連のみ |
| 広告宣伝費 | SNS広告、PR費用 | ✅ 全額 |
| 外注費 | デザイナー、開発委託 | ✅ 全額 |
| 書籍・研修費 | 技術書、セミナー | ✅ 業務関連のみ |
| 地代家賃 | 自宅オフィス | ✅ 按分（業務使用面積） |

## 重要な注意事項

1. **税務は専門家相談を推奨**
   - このシステムは整理・準備のためのツール
   - 複雑な判断は税理士に相談

2. **記録の保管**
   - 領収書・請求書は7年間保管義務
   - デジタル化推奨（e-Tax対応）

3. **期限厳守**
   - 確定申告期限を過ごすとペナルティ
   - 早めの準備を心がける

4. **法改正に注意**
   - インボイス制度等、定期的に制度変更
   - 最新情報を常にチェック
